# GroupDocs.Annotation - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Annotation](https://docs.groupdocs.com/annotation/) static documentation.

## About GroupDocs.Annotation

[GroupDocs.Annotation](https://products.groupdocs.com/annotation/) is a document annotation SDK that enables you to add comments, highlights, markups, and other annotations to PDF, Word, Excel, PowerPoint, and image files.

Available for .NET and Java.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/annotation/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
